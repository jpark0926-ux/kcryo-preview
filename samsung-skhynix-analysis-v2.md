## 3. 메모리 완판 + 사후정산 구조의 마진 영향 분석

### 🔥 구조적 변화 3대 축

#### (1) 2026년까지 메모리 완판
```
기존: 공급 과잉 → 가격 하락 → 재고 조정 → 감산 → 회복 (3-4년 사이클)
현재: AI 수요 폭증 → 2026년까지 완판 → 공급자 우위 → 가격 결정권 확보
```

- DRAM/NAND 모두 주요 고객사(빅테크)와 장기 물량 확보 경쟁
- **공급 < 수요** 구조가 최소 2027년까지 지속 전망
- 전통적 메모리 사이클론 무력화

#### (2) '사후정산' 조항 도입
```
기존 계약: 연초 가격 확정 → 연간 고정가 납품
신규 계약: 계약 종료 시점의 시장가로 추가 청구 가능
```

**마진 영향:**
- 가격 상승기: 추가 수익 확보 (업사이드 캡처)
- 가격 하락기: 기존 계약가 보호 (다운사이드 헤지)
- **비대칭적 수익구조** → OPM 상방 확대, 하방 제한
- 추정 OPM 추가 기여: +2~5%p

#### (3) 계약 기간 축소 (연단위 → 수주/월 단위)
```
의미: 시장가 반영 주기 단축 → 실시간 가격 결정권
```

- 스팟 시장 비중 확대와 유사한 효과
- 가격 상승 트렌드에서 즉각적 수혜
- 메모리 업체의 **pricing power** 역사적 최고 수준

## 4. 칩플레이션(Chipflation) 시나리오 상세 분석

### 칩플레이션이란?
> 반도체 칩 가격이 구조적으로 상승하는 현상. 원자재 인플레이션과 유사하나, **수요 견인 + 공급 제약 + 기술 프리미엄**이 동시에 작용.

### 4대 드라이버

#### 🔴 Driver 1: AI 수요의 비선형 증가
- AI 모델 파라미터: GPT-4(1.8T) → GPT-5(10T+) → 메모리 수요 5-10x
- 데이터센터 HBM 수요: 2024 $16B → 2026E $60B+ → 2027E $100B+
- **서버 1대당 메모리 탑재량**: 기존 대비 8-16배 증가

#### 🔴 Driver 2: HBM4 기술 프리미엄
- HBM3E → HBM4: 대역폭 2배, 용량 2배
- **ASP 프리미엄**: HBM4는 일반 DRAM 대비 10-15배 고가
- SK하이닉스 HBM 시장점유율: ~53% (독점적 1위)
- 삼성전자 HBM 캐치업 가속 → 전체 시장 ASP 상승

#### 🔴 Driver 3: 공급 증설 제약
- EUV 장비 리드타임: 18-24개월
- 신규 팹 건설: 2-3년 소요
- **CAPEX 투자 → 실제 공급 반영까지 시차** → 2027년까지 구조적 공급 부족

#### 🔴 Driver 4: 계약 구조 변화
- 사후정산 + 단기 계약 → 가격 상승분 즉시 반영
- 과거: 연간 고정가 → 업사이드 제한
- 현재: 시장가 연동 → **가격 상승의 100% 캡처**

### 칩플레이션 시나리오별 영향

| 시나리오 | DRAM ASP 변화 | 삼성 OPM | SK OPM | 확률 |
|---------|-------------|---------|--------|------|
| Base | +15-20% YoY | 17-19% | 52-55% | 40% |
| Bull (칩플레이션) | +30-40% YoY | 22-25% | 58-62% | 35% |
| Super Bull | +50%+ YoY | 28-30% | 65%+ | 15% |
| Bear (조정) | +5-10% YoY | 14-16% | 45-48% | 10% |

> 💡 **핵심 인사이트**: Bear 시나리오에서도 마이너스 성장이 아닌 +5-10% 성장. 이것이 구조적 슈퍼사이클의 본질.
