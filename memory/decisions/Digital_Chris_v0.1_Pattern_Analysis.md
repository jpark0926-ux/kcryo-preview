## 📊 Digital Chris 모델 v0.1

### 예측 가능한 행동
```python
if "URGENT" in subject or "수소밸브" in body:
    priority = "CRITICAL"
    response_time = "< 4 hours"
    action = "immediate_review"

elif "PO#" in subject and "Luxfer" in from_domain:
    priority = "HIGH"
    response_time = "< 24 hours"
    action = "check_inventory_confirm"

elif "견적" in subject and "Holy" in from_domain:
    priority = "HIGH"
    response_time = "24-48 hours"
    action = "delegate_to_DiKang"

elif "Meeting" in subject and time.hour in [14, 15, 16]:
    priority = "MEDIUM"
    response_time = "< 48 hours"
    action = "schedule_if_available"

elif weekday in ["Saturday", "Sunday"]:
    priority = "LOW"
    response_time = "Monday"
    action = "defer_to_weekday"
```

### 자동화 권장 사항
1. **PO 발주 알림**: Luxfer/Thunderbird PO 오면 즉시 알림
2. **긴급 키워드 필터**: "단종", "재고부족", "긴급" 등
3. **OEM 고객 우선순위**: 현대/기아 메일은 항상 HIGH
4. **주말 자동응답**: "월요일 확인 후 답변드리겠습니다"
5. **협상 타임라인 추적**: 10+ 메시지 스레드는 자동 팔로업 알림
