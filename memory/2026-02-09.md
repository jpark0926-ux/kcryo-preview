## 네이버 에디터 자동화 교훈
- DOM 직접 삭제(.remove()) → 에디터가 전체 본문 날림 ❌
- 클립보드 paste (HTML) → 본문 입력은 잘 됨 ✓
- 제목 영역은 포커스 잡기 어려움 — title click → Enter로 본문 이동 후 paste
- 여러 번 paste하면 중복/순서 문제 발생 → 한번에 전체 넣는 게 나음
