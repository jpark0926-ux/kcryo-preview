## 1️⃣ Constellation Energy (CEG)

### 방법 1: PER 기반

> **적정주가 = Forward EPS × 적정 PER**

전력/유틸리티 업종 평균 PER ~18x, 핵발전 프리미엄 + 고성장 감안.

| 시나리오 | 적정 PER | 계산 | 적정주가 |
|---------|---------|------|---------|
| 🔴 보수적 | 20x | $11.39 × 20 | **$227.80** |
| 🟡 적정 | 25x | $11.39 × 25 | **$284.75** |
| 🟢 낙관적 | 30x | $11.39 × 30 | **$341.70** |

→ PER 기반 적정가 (중간): **$284.75**

### 방법 2: PEG Ratio 기반 (피터 린치)

> **PEG = 1.0 → 적정 PER = EPS 성장률(%)**
> **적정주가 = EPS × EPS성장률(%)**

```
적정주가 = $11.39 × 21 = $239.19
```

→ PEG 기반 적정가: **$239.19**

### 방법 3: EV/EBITDA 기반

> CEG 2026E EBITDA ~$5.5B 추정, 발행주식 ~315M
> 유틸리티 EV/EBITDA 평균 ~12x, 핵발전 프리미엄 14x 적용
> 순차입금 ~$5B 가정

```
Enterprise Value = EBITDA × EV/EBITDA = $5.5B × 14 = $77.0B
Equity Value = $77.0B - $5.0B = $72.0B
적정주가 = $72.0B / 315M = $228.57
```

→ EV/EBITDA 기반 적정가: **$228.57**

- EV/EBITDA 및 DCF 추정치는 공개 컨센서스가 아닌 간이 추정치
