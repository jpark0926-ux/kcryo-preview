## 8. 경쟁 분석

### 8.1 현재 LPG 전환 블로그 시장 현황

| 유형 | 특징 | 약점 |
|------|------|------|
| **개인 후기 블로그** | 단발성 전환 후기, 높은 진정성 | 전문성 부족, 업데이트 없음 |
| **업체 홍보 블로그** | 가격·시공 사례 위주, 광고성 | 신뢰도 낮음, 정보 깊이 부족 |
| **자동차 커뮤니티 글** | 다양한 의견, 활발한 토론 | 정리 안 됨, 검증 안 된 정보 |
| **뉴스/기사** | 신뢰성 높음 | 심층 정보 부족, 실용성 낮음 |

### 8.2 로턴 블로그 차별화 포인트

| 경쟁 블로그 한계 | 로턴 블로그 차별화 |
|-----------------|-------------------|
| 광고 냄새 나는 톤 | 제3자 정보성 톤, 장단점 솔직 서술 |
| 단발성 콘텐츠 | 체계적 주 1회 발행, 시리즈 구성 |
| 기술 정보 부족 | Prins VSI-3 DI, WFS 등 기술 심화 콘텐츠 |
| 가격 정보 모호 | 가격 범위 투명 공개 (230만~540만원) |
| 보험·보증 언급 없음 | 현대해상 PL보험 10억 등 안전장치 명시 |
| OEM 관계 미언급 | KGM OEM 계약 등 공식 파트너십 강조 |
| 데이터 없는 주장 | 실제 연료비 데이터, 시뮬레이션 표 제공 |

### 8.3 차별화 실행 전략

1. **"솔직함이 최고의 마케팅"** — 바이퓨얼이 안 맞는 경우도 명시 (짧은 주행거리, 곧 폐차 예정 등)
2. **데이터 우선** — 모든 비용 비교에 계산 근거 명시
3. **시리즈 콘텐츠** — 차종별 가이드, FAQ 시리즈 등으로 재방문 유도
4. **검색 독점** — 롱테일 키워드([차종] + LPG 전환)를 차종별로 선점

## 9. 콘텐츠 제작 가이드라인

### 9.1 톤 & 보이스

| 항목 | 가이드 |
|------|--------|
| **시점** | 제3자 정보 전달자 ("~입니다", "~한다고 합니다") |
| **톤** | 친근하지만 전문적, 설명하듯 자연스럽게 |
| **금지 톤** | 광고성 ("지금 바로 전환하세요!"), 과장 ("무조건 이득!"), 가짜 후기체 |
| **솔직함** | 단점·한계도 명시 → 오히려 신뢰 상승 |
| **용어** | 바이퓨얼 (○) / 바이퓰 (✕) / 전문 용어는 괄호 설명 병기 |

#### 톤 예시

```
✅ 좋은 예:
"바이퓨얼 전환 비용은 차종과 시스템에 따라 230만원에서 540만원 사이입니다.
솔직히 적은 금액은 아니죠. 그래서 '내 주행 패턴에서 정말 본전을 뽑을 수 있는지'
따져보는 게 중요합니다."

❌ 나쁜 예:
"놀라운 가격! 단돈 230만원으로 연료비를 반으로 줄이세요!
지금 바로 상담 신청하세요!!"
```

### 9.2 포맷 구조

```
[포스트 구조 템플릿]

📌 제목 (40자 이내, 키워드 앞배치)
├── 도입부 (200자) — 독자 공감 + 핵심 키워드
├── 목차 (선택)
├── H2 소제목 1 — 핵심 정보
│   ├── 본문 + 이미지/표
│   └── 소결론
├── H2 소제목 2 — 상세 설명
│   ├── 본문 + 이미지/표
│   └── 소결론
├── H2 소제목 3 — 부가 정보
├── 정리/요약 — 핵심 포인트 3줄 정리
├── CTA — 부드러운 행동 유도
└── 태그 (5~7개)
```

### 9.3 이미지 가이드

| 항목 | 기준 |
|------|------|
| **수량** | 포스트당 최소 5장, 이상적 8~10장 |
| **유형** | 자체 촬영 우선, 인포그래픽, 비교표 캡처 |
| **크기** | 가로 860px 이상 (네이버 블로그 최적) |
| **텍스트** | 이미지 내 핵심 정보 텍스트 삽입 (모바일 가독성) |
| **워터마크** | 로턴 로고 워터마크 우측 하단 (은은하게) |
| **alt 텍스트** | 키워드 포함 설명문 필수 입력 |
| **금지** | 타사 로고 무단 사용, 저화질 이미지, 과도한 포토샵 |

### 9.4 CTA (Call-to-Action) 가이드

> CTA는 "영업"이 아닌 "안내"

```
✅ 좋은 CTA:
"바이퓨얼 전환에 대해 더 궁금한 점이 있으시면
아래 링크에서 상담 예약을 하실 수 있습니다."

"내 차종의 정확한 전환 비용이 궁금하다면,
전화 상담(031-XXX-XXXX)으로 문의해 보세요."

❌ 나쁜 CTA:
"지금 바로 전환하세요!!!"
"한정 할인 중! 서두르세요!"
```

**CTA 배치 원칙:**
- 포스트 하단 1회만 (본문 중간 삽입 금지)
- 자연스러운 문맥 연결
- 연락처·링크는 텍스트로 (배너 이미지 지양)

### 9.5 내부 링크 전략

- 모든 포스트에 관련 이전 포스트 1~2개 링크
- "관련 글 더 보기" 섹션으로 정리
- 시리즈물은 이전/다음 편 명시

```
📎 관련 글
- LPG 바이퓨얼이 뭔가요? → [링크]
- 전환 비용 총정리 → [링크]
```
