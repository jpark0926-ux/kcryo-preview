# Prins VSI-3 DI 시스템이란? 유럽 인증 직분사 LPG의 모든 것

직분사(GDI) 엔진 차량에 LPG를 적용할 수 있을까? 과거에는 불가능에 가까웠지만, 네덜란드의 Prins사가 개발한 **VSI-3 DI 시스템** 덕분에 지금은 현실이 되었습니다. 오늘은 유럽에서 인증받은 직분사 LPG 시스템, Prins VSI-3 DI에 대해 상세하게 알아봤습니다.

---

## Prins는 어떤 회사?

**Prins Autogassystemen**은 1986년 네덜란드에서 설립된 LPG/CNG 자동차 연료 시스템 전문 기업입니다. 현재는 글로벌 대체 연료 기업인 **WFS(Westport Fuel Systems)**의 일원으로, 전 세계 50개국 이상에 시스템을 공급하고 있습니다.

Prins가 주목받는 이유는 단순한 LPG 키트 제조사가 아니라, **직분사 엔진에 특화된 LPG 시스템**을 세계 최초로 양산한 기업이기 때문입니다.

### Prins 연혁 주요 포인트
- 1986년: 네덜란드에서 설립
- VSI-1, VSI-2: MPI(포트 분사) 엔진용 시스템 개발
- **VSI-3 DI**: 직분사(GDI/TGDI) 엔진 전용 시스템 출시
- WFS(Westport Fuel Systems) 그룹 편입
- 전 세계 50개국 이상 공급

---

## VSI-3 DI란? 직분사 LPG의 핵심

**VSI-3 DI(Vapour Sequential Injection - 3rd Generation, Direct Injection)**는 직분사 엔진에 LPG를 적용하기 위해 개발된 3세대 순차 분사 시스템입니다.

기존 MPI 엔진용 LPG 시스템과의 가장 큰 차이점은, **직분사 엔진의 고압 분사 환경에 맞춰 설계**되었다는 것입니다. 직분사 엔진은 연료를 실린더 안에 직접 분사하기 때문에 훨씬 정밀한 제어가 필요한데, VSI-3 DI는 이 요구 사항을 충족합니다.

---

## 핵심 구성 부품 상세 분석

VSI-3 DI 시스템은 여러 핵심 부품들로 구성되어 있으며, 각각이 국제 인증을 받은 고품질 부품입니다. 하나씩 살펴보겠습니다.

### 1. ECU — AFC-3.0 DI (두뇌)

| 항목 | 내용 |
|------|------|
| 명칭 | AFC-3.0 DI |
| 역할 | LPG 분사량·타이밍 제어, 가솔린 ECU와 실시간 통신 |
| 인증 | **EMC E4-10R** (유럽 전자파 적합성 인증) |
| 특징 | 기존 가솔린 ECU를 방해하지 않으면서 독립적으로 LPG 제어 |

EMC E4-10R 인증은 ECU가 차량 내 다른 전자장치에 전자파 간섭을 일으키지 않음을 의미합니다. 현대 차량은 수십 개의 ECU가 동시에 작동하므로, 이 인증이 없는 ECU는 오작동 위험이 있을 수 있습니다.

### 2. 감압기 — eVP-500 (심장)

| 항목 | 내용 |
|------|------|
| 명칭 | eVP-500 |
| 역할 | 액체 LPG를 기체로 변환, 일정 압력으로 공급 |
| 등급 | **500HP Class 1** |
| 특징 | 전자식 압력 제어, 최대 500마력급 엔진 대응 |

Class 1은 감압기의 최고 등급입니다. 500마력급까지 대응 가능하다는 것은 국내 대부분의 승용차·SUV에 여유 있게 적용할 수 있음을 의미합니다. 전자식으로 압력을 정밀 제어하기 때문에 엔진 부하 변화에도 안정적인 연료 공급이 가능합니다.

### 3. 인젝터 — Keihin KN-9 (손끝)

| 항목 | 내용 |
|------|------|
| 명칭 | Keihin KN-9 |
| 역할 | LPG를 흡기 포트에 정밀 분사 |
| 내구성 | **24만 km** 테스트 통과 |
| 제조사 | Keihin (일본, Honda 그룹 계열) |

Keihin은 Honda 그룹 계열의 정밀 부품 제조사로, 자동차 인젝터 분야에서 세계적인 기술력을 보유하고 있습니다. KN-9 인젝터가 24만 km 내구성을 검증받았다는 것은, 일반적인 승용차 사용 기간(10~15년) 동안 인젝터 교체 없이 사용 가능하다는 의미입니다.

### 4. 연료 라인 — XD-Line (혈관)

| 항목 | 내용 |
|------|------|
| 명칭 | XD-Line |
| 등급 | **Class 1** |
| 특징 | 고압 LPG 전용, 누출 방지 설계 |

연료 라인은 눈에 잘 띄지 않지만 안전과 직결되는 부품입니다. Class 1 등급의 XD-Line은 고압 LPG 환경에서도 누출 없이 안정적으로 연료를 공급합니다.

### 5. 필터 시스템 (면역 체계)

| 항목 | 내용 |
|------|------|
| 불순물 차단율 | **99%** |
| 역할 | LPG 내 이물질·수분 제거 |
| 교체 주기 | 2~3만 km마다 권장 |

LPG에 포함된 미세한 이물질과 수분은 인젝터 막힘이나 감압기 고장의 원인이 될 수 있습니다. 99% 불순물 차단 필터는 이런 문제를 사전에 방지하여 시스템 전체의 수명을 연장합니다.

## R115 인증이란?

**R115(UN ECE Regulation 115)**는 유럽경제위원회(UNECE)에서 제정한 **LPG/CNG 개조 차량에 대한 국제 안전·성능 규정**입니다.

R115 인증을 받으려면 다음 항목을 모두 통과해야 합니다:

| 인증 항목 | 기준 |
|-----------|------|
| 출력 변화 | 가솔린 대비 출력 손실 허용 범위 이내 |
| 배기가스 | 기존 배기 인증 기준 충족 |
| 안전성 | 연료 시스템 누출·내압·충격 테스트 |
| 내구성 | 장기 사용 시 성능 유지 검증 |

Prins VSI-3 DI 시스템은 R115 인증에서 **출력 손실 0%, CO₂ 배출 10% 감소**라는 결과를 기록했습니다. 이는 LPG로 전환해도 가솔린과 동일한 출력을 유지하면서, 환경적으로는 더 나은 성능을 발휘한다는 것을 의미합니다.

## 직분사 LPG는 일반 LPG 개조와 뭐가 다를까?

| 비교 항목 | 믹서 방식 (1세대) | MPI 순차분사 (2세대) | 직분사 VSI-3 DI (3세대) |
|-----------|-------------------|---------------------|------------------------|
| 분사 방식 | 공기와 혼합 후 공급 | 흡기 포트에 개별 분사 | 직분사 엔진 전용 제어 |
| 출력 손실 | 10~15% | 3~5% | **0%** (R115 인증) |
| 적용 엔진 | 캬뷰레터/초기 MPI | MPI 엔진 | **GDI/TGDI 직분사 엔진** |
| ECU 제어 | 단순/없음 | 기본 제어 | 가솔린 ECU 실시간 연동 |
| 국제 인증 | 없음 | 일부 | **R115, EMC 등 다수** |

## 국내에서는 어디서 시공 가능할까?

Prins VSI-3 DI 시스템은 Prins/WFS의 **공식 파트너사**를 통해서만 정식 시공이 가능합니다. 국내에서는 **로턴(ROTURN)**이 WFS의 공식 파트너로서 이 시스템을 보급하고 있습니다.

로턴 관련 참고 정보:
- 2013년 설립, 본사 용인
- 모회사: 한국초저온(주)
- **국내 최초 LPG 직분사 보급**
- KGM(구 쌍용) 공식 공급 계약 — 토레스, 액티언, 코란도, 티볼리 등 5개 모델
- 현대해상 PL보험 최대 10억 원
- 카카오블랙, 점보택시에도 적용
- 개조 비용: 차종에 따라 230만~540만 원 (VAT, 충전비, 검사수수료, 마감재 포함)

더 자세한 내용은 로턴 공식 홈페이지(roturn.com)나 네이버 카페(cafe.naver.com/gdilpg)에서 확인하실 수 있습니다.

## 마무리

Prins VSI-3 DI 시스템은 직분사 엔진 시대에 맞춘 3세대 LPG 기술입니다. R115 인증을 통해 출력 손실 0%와 CO₂ 10% 감소가 검증되었고, 각 부품이 국제 인증과 내구성 테스트를 거쳤다는 점에서 기술적 신뢰성이 높은 시스템이라 할 수 있겠습니다.

직분사 차량의 LPG 개조를 고려 중이시라면, 시스템의 인증 등급과 부품 스펙을 꼼꼼히 비교해 보시길 권합니다.
