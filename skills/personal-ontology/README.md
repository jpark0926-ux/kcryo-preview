# Personal Ontology Skill

Transform OpenClaw into your Personal Operating System with Palantir-style ontology.

## Quick Start

1. **Copy template**:
   ```bash
   cp ontology-template.yml ~/workspace/ontology.yml
   ```

2. **Edit with your data**:
   - Projects, companies, investments
   - Goals, habits, energy patterns
   - Priorities and blockers

3. **Make scripts executable**:
   ```bash
   chmod +x ontology-*.sh daily-summary.sh
   ```

4. **Test status check**:
   ```bash
   ./ontology-status.sh
   ```

5. **Set up automation** (optional):
   - Daily briefing at 09:00
   - Weekly insights on Friday 19:00
   - See SKILL.md for cron examples

## Files

- `SKILL.md` - Complete documentation
- `ontology-template.yml` - Starting point for your ontology
- `ontology-status.sh` - Instant status check (<1sec)
- `ontology-query.py` - Query interface (requires PyYAML)
- `ontology-sync.py` - Auto-sync from READMEs (requires PyYAML)
- `daily-summary.sh` - Morning briefing generator
