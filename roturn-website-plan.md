## 2. 경쟁사 벤치마킹

### 2-1. Prins Autogas (prinsautogas.com) — 글로벌 1위
- **강점**: 35년 이상 경력 강조, 제품 라인업별 상세 페이지, DI 엔진 전문성 어필
- **구조**: 제품별 페이지 + 딜러 네트워크 + 뉴스/블로그
- **시사점**: "전문성 + 역사"로 신뢰 구축, 제품별 기술 스펙 상세 제공

### 2-2. Prins Turkey (prins.com.tr) — 지역 딜러 모범 사례
- **강점**: EcoMax/Technomax/VSI-2/VSI-3 등 제품별 상세 페이지, R67-01 인증 표시, 각 제품에 "İncele(살펴보기)" CTA
- **구조**: 홈(제품 카드형 나열) → 제품 상세 → 연락처
- **시사점**: 제품을 카드형으로 나열하면 직관적, 인증마크 활용, 각 제품에 명확한 CTA

### 2-3. Lovato Gas (lovatogas.com) — 글로벌 브랜드
- **강점**: Landi Renzo 그룹 소속 신뢰감, 글로벌 네트워크, B2B + B2C 동시 타겟
- **구조**: 제품 카테고리 + 딜러 찾기 + 회사 소개
- **시사점**: 모회사/파트너십 활용한 신뢰 구축

### 벤치마킹 핵심 시사점
| 요소 | 경쟁사 공통점 | 로턴 적용 |
|------|-------------|----------|
| 신뢰 구축 | 인증마크, 시공 실적, 역사 강조 | 인증/실적 섹션 신설 |

### 브랜드 보이스
- **전문적이되 쉬운**: 기술 용어는 쓰되, 항상 쉬운 설명을 덧붙임
- **신뢰감 있는**: 수치와 데이터로 뒷받침
- **고객 중심**: "우리 기술이 좋다"보다 "당신이 얼마나 절약하는지"
- **행동 유도**: 모든 섹션 끝에 다음 행동을 안내

### 카피 공식
```
[문제 인식] → [해결책 제시] → [증거/신뢰] → [행동 유도]

예시:
매달 빠져나가는 유류비, 부담되시죠?          ← 문제
로턴 바이퓨얼 시스템으로 연료비 40% 절감.    ← 해결
시공 3,000대+, 고객 만족도 98%.             ← 증거
지금 무료 상담 받아보세요.                   ← 행동
```
